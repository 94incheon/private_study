### 재귀를 이용하여 순열 만들기
> 동철이 문제에 활용됨 (D4)
> 경우의 수는 3! (3팩토리얼) 이 된다.

```python
def f(n, k): # 순열의 n번 원소 결정
    if n == k:
        print(p)
    else:
        for i in range(k):
            if used[i] == 0: # i번 원소가 사용되지 않았으면
                used[i] = 1 # 사용함으로 표시
                p[n] = A[i]
                f(n+1, k) # n+1 원소 결정
                used[i] = 0 # 다른 자리에서 사용하도록 풀어줌

# 순열을 생성하는 코드... (이해할것, 외우면 응용을 못함)
A = [1,2,3]
used = [0] * len(A)
p = [0] * len(A)

f(0, 3)
