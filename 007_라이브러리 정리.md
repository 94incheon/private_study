## Collections

- `Counter`
  
  - counter(iterable)
  
  - ```python
    import collections
    my_list = [1, 2, 3, 4, 5, 6, 7, 8, 7, 9, 1, 2, 3, 3, 5, 2, 6, 8, 9, 0, 1, 1, 4, 7, 0]
    answer = collections.Counter(my_list)
    
    print(answer[1]) # = 4
    print(answer[3])  # = 3
    print(answer[100]) # = 0
    ```
- `most_common`
  
  - 튜플형태의 리스트를 반환함



```python
T = int(input())

for tc in range(1, T+1):
    N = int(input())
    d = list(input())
    c = Counter(d).most_common()
    print(c)
```
