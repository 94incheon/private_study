## Collections

- `Counter`
  - counter(iterable)
- `most_common`
  - 튜플형태의 리스트를 반환함



```python
T = int(input())

for tc in range(1, T+1):
    N = int(input())
    d = list(input())
    c = Counter(d).most_common()
    print(c)
```
